常用SHELL命令

mkdir 创建文件夹

mkdir -p test/file
rmdir -p test/file
-p 一次性创建/删除多级目录

cd 改变工作目录

cd /home
~默认为/home路径

pwd 显示当前目录的绝对路径

pwd

ls 显示目录的内容的命令

ls -al /etc
选项
-a 显示所有文件和子目录
-l 显示文件和子目录的详细信息
-d 如果参数是目录,则只显示目录的信息,不显示其所包含文件的信息
-t 按时间顺序显示
-R 递归显示各子目录中文件和子目录信息

cat 显示文件内容

cat > f1 /// 此时按”Ctrl+D”可将f1文件保存在当前目录下
echo ‘abcde’ > f1 //写点啥
cat f1 //查看文件内容

more 查看内容较多文件,可以分屏显示内容

more /etc/passwd
Enter/space 翻页
b 向前翻
q 退出

less 同 more

区别:more命令可以查看二进制文件,less只能查看ASCII码文件

head 显示文件头几行内容

head -5 /etc/passwd

tail 同head

tail -5 /etc/passwd

grep 文件内容查询

grep ‘root’ /passwd
往往配合管道使用
head -50 /etc/passwd | grep data *

find 文件查找命令(自带递归)

从指定的目录开始,递归搜索各个子目录
-name 查找文件名与所给字符串匹配的所有文件,字符串内可用通配符*,?,[]
-group 查找属于某个用户所属组的所有文件
-user 查找属于某个用户的所有文件
-type 根据文件的类型进行查找
-a 逻辑与 -o 逻辑或 !逻辑非

find / -name ‘temp’ -o -name ‘install*’
find / -group root

wc 文件内容统计命令

-c 统计字节数
-l 统计行数
-w 统计字数
wc -clw /etc/fstab

cp 文件的复制,移动,和删除命令

-b 如果存在同名文件,覆盖前备份原来的文件 ~f2为备份文件
-f 强制覆盖同名文件
-r/-R 递归复制,同时复制其子目录和文件

cp -r f1 f2
f1为源文件 f2为目标文件 linux中对两个目标操作时都按照源在左，目标在右的方式

mv 移动或重命名文件或目录

-b 如果存在同名文件,覆盖前备份原来的文件
-f 强制覆盖同名文件
mv f1 test //将文件f1移动到test目录下
mv test mytest//将test重命名为mytest

rm 删除文件或目录

-f 强制删除,不出现确认信息
-r/-R 递归删除目录

rm -r mytest

man 查看手册

man mkdir

clear 清屏

BASH自动补齐

例如输入mk,连按两次tab键,屏幕上会显示所有以mk开头的命令
按一次tab键,可以自动补全。

history 历史命令记录

history -n //参数可带可不带,查看前n行命令记录
!n//执行序号为n的命令

alias命令别名

查看当前用户可使用的别名

alias

给某个命令设置别名

alias llt=’ls -l test’

通配符

1.* 匹配任意个数的任意字符
2. ? 匹配单个任意字符
3. [acd]/[a-g] 匹配[]范围内的某个字符

重定向

ls -a > info

重定向符号和cat命令结合使用

创建一个文件,Ctrl+D结束输入

cat > file1 cat > file2

合并两个文件

cat file1 file2 > newfile

向文件中追加信息

cat >> newfile

错误信息重定向

ls /test 2>error

cat f1 | grep ‘file’ | wc -l

磁盘的分类 (根据连接的接口种类分类)

  • IDE磁盘
  • SCSI磁盘
  • 软盘
  • 移动磁盘

    IDE磁盘

  • ATA接口的IDE磁盘

    ATA接口的IDE磁盘设备识别名称为hd
    linux会为每一个硬盘提供一个英文字母代号,作为每个不同的ATA IDE磁盘的识别名称,例如,第一个ATA IDE磁盘的设备文件是/dev/hda,第二个是/dev/hdb

  • SATA接口的IDE磁盘

    设备文件识别名称是sd

    SCSI磁盘

    SCSI磁盘是使用SCSI接口连接到计算机的磁盘,由SCSI控制卡上独立的处理器执行调用磁盘的动作,比起IDE是由主机板上的CPU处理的情况而言
    一台计算机可以安装多个SCSI控制卡,每一个SCSI控制卡中可以安装数个SCSI磁盘
    SCSI磁盘代号有两个字母 /dev/sda表示第一个SCSI磁盘,/dev/sdad表示第30个SCSI磁盘

磁盘名称如何分配

  • 启动系统时指定
  • SCSI控制卡的顺序
  • SCSI磁盘设置的序号

进程

进程和程序的区别

进程:一个程序在其自身的虚拟地址空间的一次执行活动,是系统资源分配和调度的基本单位
程序:存储在磁盘上包括可执行机器指令和数据的静态实体

进程的类型

  1. 交互进程:由一个shell启动的进程
  2. 批处理进程
  3. 守护进程：在后台运行而又没有终端或登录shell与之结合在一起的进程

    启动进程 1.手工启动 2.调度启动
    手工启动 1.前台启动 2.后台启动

    前台启动

    当用户输入一个命令,就会启动一个前台进程
    这里启动的前台进程相当于当前shell进程的子进程,由于子进程结束后才能继续父进程,此时用户不能进行其他操作。
    cat f1 | grep ‘file’ | wc -l 这条命令由三条语句组成,所以启动了三个子进程

    后台启动

    在命令的后面加上 & 如下
    ls -a > text &
    此时启动了一个后台子shell进程,此时父进程不需要等待子进程结束就可以继续运行

    后台进程的应用场景

  4. 该程序运行期间不需要用户的干预
  5. 该程序执行时间较长
