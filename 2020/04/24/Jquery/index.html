1jQuery简介

1.1jQuery简介

什么是jQuery：JavaScript+查询（Query），是由javascript编写的js库，提供了更高效便捷的接口提高开发效率。
jQuery的特点：
jQuery是一个轻量级的javascript框架，核心理念是WRITE LESS,DO MORE。
jQuery应用广泛，在世界前10000个访问最多的网站中，有超过55%在使用jQuery。
jQuery是免费、开源的，文档丰富。
众多第三方的js库都采用jQuery辅助编写，如jquery-mobile、easyUI、jqueryUI等等。

jQuery的版本：
目前jQuery已经升级到3.0+版本。本次课程的版本jquery1.9.1
jQuery的中文网站地址如下，可以下载中文文档以及源码

http://www.css88.com/jqapi-1.9/

1.2jQuery功能介绍

jQuery的核心功能如下：
HTML 元素选取：提供了对dom元素的选取方式。
HTML 元素操作：提供了对dom元素的操作功能。
CSS 操作 ：提供了对dom元素样式的修改功能。
HTML 事件函数 ：扩展了javascript事件，并实现了兼容
JavaScript 特效和动画
HTML DOM 遍历和修改
AJAX ：简化了javascript对于ajax的调用。
Utilities ：提供了多种工具。

jQuery的基本语法如下：

例如以下代码实现网页全部段落标签的隐藏功能。

$("p").hide();

1.7核心函数

jQuery：是jQuery的核心函数，可以缩写为$。主要功能：
获取DOM对象：获取文档树上dom对象(jQuery封装后的)。
创建元素：创建html元素并插入到DOM树上。
克隆对象：创建一个jQuery对象的副本。
源码解析：jQuery函数，第一个参数为jQuery的选择器，第二个参数为DOM对象，jQuery会判断传递参数的类型。

2jQuery对象

2.1jQuery对象

jQuery 对象：就是通过jQuery封装的DOM对象，提供了兼容性更强方法以及属性。


使用jQuery对象的优缺点：
优点：jQuery的扩展属性和函数在底层判断了浏览器以及版本实现了浏览器的兼容。
缺点：jQuery扩展的函数只能实现基本的dom操作，但是对于元素独有的htmldom接口封装很少(例如 input、select())，对于html5新增的DOM方法也不支持，如果想实现类似功能，只能使用dom接口。
结论：为保证兼容性尽量使用jQuery对象的API，但特殊功能也需要使用原生dom接口。
编码习惯：如果获取的是 jQuery 对象, 那么要在变量前面加上 $。
var $variable = jQuery 对象
var variable = DOM 对象

2.2DOM对象与jQuery对象的转换

将DOM对象转换为jQuery对象：用$()把DOM对象包装起来即可。

//DOM对象转换为jquery对象的方式
var div = document.getElementById('div') ;
var $div = $(div);
$div.html('使用jquery对象操作');

将jQuery对象转换为DOM对象:一个jQuery对象可以封装多个DOM对象。通过size()方法，返回选择器选择的dom对象的数量。
通过下标获取DOM对象。

/ *使用数组下标获取dom对象* /
var firstInput =$input[0];

通过get(index)方法，得到相应的DOM对象

//使用get方法获取
var firstInput = $input.get(0);

3jQuery选择器

3.1选择器的使用方式

jQuery选择器：用于获取jQuery对象的表达式。
基本语法：$(“选择器表达式”)，例如$(“p”)

选择器的或运算：用来获取符合多个表达式的jQuery对象。

jQuery(“selector1, selector2, selectorN” );
jQuery(“#id,div,class”);
//“#id”、”div”、”.class” 为不同的选择器表达式。

3.2基本选择器

基本选择器是 jQuery 中最常用的选择器, 通过元素的id, 引用的样式类名、标签名、通配符来查找 DOM 元素。
id选择器:选择一个具有指定id属性的单个元素。

示例：$(”#ID”);

元素选择器: 参数为html标签名，返回当前网页的该标签对应的jQuery对象。

示例： $(”div”) ；

类选择器:参数为css的样式类名，返回使用当前样式类渲染的全部元素对应的jQuery对象。

示例： $(”.myClass”)；

通配符选择器：匹配所有元素对应的jQuery对象。

示例： $(”*”)；

基本选择器大家比较熟悉，这里不作介绍。

3.3属性选择器

属性选择器：通过元素的属性来获取相应的元素。
[attribute]： 匹配包含指定属性的元素。
$(”div[id]“) 获取所有具有id值的div对象。
[attribute=value]：匹配给定的属性是某个特定值的元素。
$(”input[name=‘userName’]“) 获取name取值为userName的输入框。
[attribute!=value]：匹配所有不含有指定的属性，或者属性不等于特定值的元素。
$(”input[name!=‘userName’]“)获取name取值为非userName或不具备name属性的输入框。
[attribute^=value]：匹配给定的属性是以某些值开始的元素
$(”input[name^=‘news’]“)获取name取值以news开头的输入框。
[attribute$=value]：匹配给定的属性是以某些值结尾的元素。
$(”input[name$=‘end’]“)获取name取值以end结尾的输入框。
[attribute=value]:匹配给定的属性是以包含某些值的元素。
$(”input[name=‘con’]“) 获取name取值包含con的输入框。
[attributeFilter1][attributeFilterN]：组合使用。
$(”input[id][name$=‘man’]“) 获取具有id属性，name取值以man结尾的输入框

    <input type="text" name="age" id="age" class="age">
</body>

$(”label + input”) 獲取label标签的下一个input标签的dom元素

3.5.2子元素过滤选择器

:nth-child(index/even/odd/equation):匹配选中元素是父元素的第（index）/基数/偶数/3n/个元素。与的 eq() 有些类似,不同的地方就是前者是从0开始,后者是从1开始。

4DOM操作

4.1创建DOM元素

DOM操作：DOM不专属于javascript，而是w3c制定的标准，定义了操作网页元素的接口，而浏览器都实现了这些接口。
DOM 操作的分类:
DOM Core: 此接口用来解析的修改任何一种使用标记语言编写出来的文档, 例如: XML，而html是古老的xml。例如document.getElementById(‘id’);
HTML DOM:此接口用来解析的修改html网页元素的接口。例如tr.insertCell(0);
CSS-DOM:此接口用来解析的修改html网页的样式,也可以看做htmldom的一类。例如tr.style.fontSzie=’10px’

创建DOM元素: 使用 jQuery 的工厂函数 $(html); 会根据传入的 html 标记字符串创建一个 jQuery的DOM 对象并返回。
注意事项:
动态创建的新元素节点不会被自动添加到文档中, 而是需要使用其他方法将其插入到文档中 例如append（）;
当创建单个元素时, 需注意闭合标签和使用标准的 XHTML 格式. 例如创建一个

4.2插入DOM元素

插入dom元素：将制定的或新创建的jQuery的dom对象插入到指定位置上。按照插入方式的不同，可分为
插入到某元素内作为该元素的子节点方式；
parent.append(child) :向父元素（parent）加入子元素（child），插入后子元素为父元素的最后一个孩子。
child.appendTo(parent) :将子元素（child）插入到父元素（parent）下，插入后子元素为父元素的最后一个孩子。与append调用方式相反，功能相同。
parent.prepend(child):向父元素（parent）加入子元素（child），插入后子元素为父元素的第一个孩子。
child.prependTo(parent) :将子元素（child）插入到父元素（parent）下，插入后子元素为父元素的第一个孩子。与prepend调用方式相反，功能相同。
parent、child一般的使用场景都是jQuery的dom对象。

插入到某元素外作该元素的兄弟节点方式：
brother.after(obj) :在某个兄弟节点（brother）之后插入目标对象（obj）。
brother.before(content):在某个兄弟节点（brother）之前插入目标对象（obj）。
obj.insertAfter(content):将目标对象（obj）在插入到某个兄弟节点（brother）之后。
obj.insertBefore(content) :将目标对象（obj）在插入到某个兄弟节点（brother）之前。
brother、obj一般的使用场景都是jQuery的dom对象。

<body>
    <ul>
        <li>北京</li>
        <li>天津</li>
    </ul>
</body>
<script>
    $(function(){
        //创建元素，一定要闭合
        var $li = $("<li>上海</li>");
        //通过父元素插入,最后
        $("ul").append($li);
        //子元素添加到父元素
        $li.appendTo($("ul"))
        //父元素将子元素添加到最前面
        $("ul").prepend($li);
        //子元素添加到复元素的最前面
        $li.prependTo($("ul"));

        //通过兄弟元素插入,插入兄弟的后面
        $("ul li:nth-child(2)").after($li);
        //元素调用
        $li.insertAfter($("ul li:nth-child(2)"));
        //插入兄弟的前面
        $("ul li:nth-child(2)").before($li);
        //当前元素调用
        $li.insertBefore($("ul li:nth-child(2)"));
    })
</script>

5jQuery的事件处理

5.1公用事件的绑定方式

jQuery对事件进行的扩展包含：
绑定事件的方式
公用事件绑定方式
特殊事件绑定方式
事件对象的扩展:对常用事件属性和方法做了浏览器兼容，通过on方法实现了事件委托机制

jQuery的老版本事件绑定机制：
jQuery的老版本提供了若干绑定以及解除事件的函数如（bind、delegate、live），但已经过时。可通过如下网站进行了解。

http://www.jquery123.com/category/events/event-handler-attachment/

jQuery的推荐事件绑定方式：
on：在选定的元素上绑定一个或多个事件处理函数。
off：移除一个事件处理函数。

on方法：是jQuery目前推荐的绑定事件的方式，特点如下
将事件绑定委托给jQuery，因此可以通过jQuery选择器选择一组dom对象统一绑定。
on方法倾向于将所有事件绑定到一个区域，当该区域内的任意一个元素触发事件时，jQuery会通过事件的冒泡机制找到该区域对应的事件函数，然后执行，减少了绑定事件的次数。
on方法可以向事件处理函数传递参数。

on方法使用：

$('outerSelector').on('eventType','childSelector',data,fun);

outerSelector:外层区域选择器。
eventType：事件类型 如click，字符类型。
childSelector：内部元素选择器（缺省为外部选择器）。
data：传递给执行函数的参数，封装在event.data中。json类型。
fun：事件处理函数
示例：向目标区域outerDiv中的全部按钮绑定事件。

$('#outerDiv').on("click",'input:button',{id:1},function(e) {
alert( 'div中的按钮点击触发'+e.data.id) ;
});

on方法省略部分参数：
省略了传递参数以及子元素选择器，其余不能省略。

$('#but').on("click",function () {
    alert('点击所对应的事件');
});

省略子元素选择器

$('#but1').on("click", {id:1}, function(e) {
    alert('传递的参数为id='+e.data.id) ;
});

off方法使用：

$('outerSelector').off('eventType','childSelector',fun);

outerSelector:外层区域选择器。
eventType：事件类型 如click，字符类型
childSelector：内部元素选择器（缺省外部选择器）。
fun：需要解除的事件处理函数
示例

$('#button4').off('click',createCheck) ;

6.3Ajax访问

jQuery封装了多个用于发送ajax请求的底层函数，我们这里只介绍轻量好用的几个。
jQuery.get()/post()：发送get或post请求，并接收服务器端返回数据。

get(url,data,function(sData,status,xhr),type):

url:访问地址,字符类型
data：发送给服务器的数据。
function：服务器成功响应的回调参数
–sData：服务器返回数据，格式根据type决定。
–stats：服务器返回状态，默认为success
–xhr：发送请求xmlHttpRequest对象
type：请求类型（xml, json, script, 或 html）

jQuery.getJSON() ：发送get请求，并接收服务器端返回数据，数据格式为JSON。

getJSON(url,data,function(sData,status,xhr)):

url:访问地址,字符类型
data：发送给服务器的数据。
function：服务器成功响应的回调参数
–sData：服务器返回数据，格式根据type决定。
–stats：服务器返回状态，默认为success
–xhr：发送请求xmlHttpRequest对象

jQuery.load() ：从指定地址读取html代码并写入到指定对象中。

getload(url,function(html,status,xhr)):

url:访问地址,字符类型
function：服务器成功响应的回调参数
–html：html代码格式的字符。
–stats：服务器返回状态，默认为success
–xhr：发送请求xmlHttpRequest对象
