javaScript

javaScript的特点(对比于java等语言)

1.弱类型变量:javaScript的变量无特定类型,用关键字var定义,可被初始化为任意的值。
2.每行结尾的分号可以不写,若不写,则默认把这行代码的结尾看作该语句的结尾。

变量

1.javaScript的变量不一定需要初始化。

2.javaScript可以在同一个变量中存储不同的数据类型。

3.javaScript可以用var同时声明多个类型的变量。

4.javaScript可以不声明变量就直接使用。

javaScript的解释程序在遇到未声明过的变量时,会自动用该变量创建一个全局变量,并将其初始化为指定的值。

数据类型

javaScript一共有9种数据类型
1.(未定义)Undefined,
2.(空)Null,
3.(布尔型)Boolean,
4.(字符串)String,
5.(数值)Number,
6.(对象)Object,
7.(引用)Reference,
8.(列表)List,
9.(完成)Completion
其中7.8.9三种类型仅仅作为javaScript运行时中间结果的数据类型,不能在代码中使用。

String

字符串里面包含双引号时应该把整个字符串放在单引号里,如下:

charAt()方法

获取指定位置的字符,第一个字符的位置为0

slice(),subString(),substr()方法

slice()和subString方法都接受两个参数,分别为字符串的起始位置和终止位置,[,),如果只传入一个参数,则默认从起始位置到字符串结尾。

第二个参数为负数时,slice()方法从后往前计数,substring()方法忽略负数,作为0处理,以较小数作为起始位,较大数作为终止位。

indexOf(),lastIndexOf()

查找某个字符所在的位置
indexOf()从前往后,lastIndexOf()从后往前,返回的数值是在字符串中的绝对位置,不是相对于查找的起始位置,若找不到则返回-1
