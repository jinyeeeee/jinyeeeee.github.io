javaScript

javaScript的特点(对比于java等语言)

1.弱类型变量:javaScript的变量无特定类型,用关键字var定义,可被初始化为任意的值。
2.每行结尾的分号可以不写,若不写,则默认把这行代码的结尾看作该语句的结尾。

变量

1.javaScript的变量不一定需要初始化。

2.javaScript可以在同一个变量中存储不同的数据类型。

3.javaScript可以用var同时声明多个类型的变量。

4.javaScript可以不声明变量就直接使用。

javaScript的解释程序在遇到未声明过的变量时,会自动用该变量创建一个全局变量,并将其初始化为指定的值。

数据类型

javaScript一共有9种数据类型
1.(未定义)Undefined,
2.(空)Null,
3.(布尔型)Boolean,
4.(字符串)String,
5.(数值)Number,
6.(对象)Object,
7.(引用)Reference,
8.(列表)List,
9.(完成)Completion
其中7.8.9三种类型仅仅作为javaScript运行时中间结果的数据类型,不能在代码中使用。
